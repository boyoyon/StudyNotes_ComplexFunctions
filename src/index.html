立体射影

画像が球面の一部に貼り付けられるだけのプログラムです

① 画像 → 立体射影 → PLYファイル書き出し
　 python stereographics.py (画像ファイル)

　「stereographics.ply」が出力されます。

入力画像例

② PLYファイルの表示
　 python o3d_display_ply.py stereographics.ply

操作機能
左ボタン押下＋ドラッグ点群の回転
ホイールボタン押下＋ドラッグ点群の移動
ホイール回転点群の拡大・縮小
PrintScreenキー押下スクリーンショット保存
ウィンドウ閉じるボタン押下　プログラム終了
